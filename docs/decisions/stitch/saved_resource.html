Voici les nouveaux écrans ajoutés au canvas :

  1. Modal Réservation - Étape 1/2 : Le formulaire initial avec un indicateur d'étape (Stepper) en haut pour guider l'utilisateur.
  2. Modal Alerte Délai Court : La fenêtre d'avertissement pour les départs immédiats, désormais présentée comme une étape intermédiaire du processus.
  3. Modal Confirmation Final : La vue de succès avec la référence de réservation, marquant la fin du parcours client.

Le workflow est désormais clairement visualisé par le "stepper" en haut de chaque modal, assurant une cohérence parfaite entre toutes les étapes de la réservation.
